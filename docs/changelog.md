# Changelog

Releases are cut automatically by [release-please](https://github.com/googleapis/release-please)
from [Conventional Commits](https://www.conventionalcommits.org/) on `main` — there is no
hand-maintained changelog file to browse here. See the
[GitHub Releases page](https://github.com/rknightion/sf2loki/releases) for the notes attached to
each tagged version, or [`CHANGELOG.md`](https://github.com/rknightion/sf2loki/blob/main/CHANGELOG.md)
in the repository for the same history in one file. See
[Release Process](development/release-process.md) for how a commit becomes a release.
